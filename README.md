### About Me

I'm a software engineer with a passion for building immersive web experiences. I work across the full stack but gravitate toward the intersection of creative development and engineering -- from interactive 3D environments with WebGL/WebGPU to full-stack TypeScript applications.

- Building spatial learning platforms and interactive 3D experiences
- Exploring shaders, WebGPU, and creative coding
- Full-stack work with TypeScript, React, SolidJS, NestJS, and more
